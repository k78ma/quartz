>[!info] Antiderivative of vector valued functions
>A vector valued function $\vec{V}(t)$ is said to be an antiderivative of $\vec{v}(t)$ on the interval $\alpha<t<\beta$ if
>$$
>\frac{ d\vec{V} }{ dt } = \vec{v}(t) \text{\; for \;} \alpha<t<\beta
>$$
